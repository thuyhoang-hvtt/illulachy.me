# dotWord — Landing Page Design

## Project Context
- **Project Name**: dotWord
- **Type**: Mobile App Landing Page
- **Tech Stack**: React Native
- **Purpose**: A mobile app that teaches you a new word every day with inspirational quotes, making vocabulary building a daily habit

## Enhanced Design Prompt

**Atmosphere**: Clean, modern, and inspirational with a focus on learning and personal growth. The vibe should be educational yet approachable, with warmth and encouragement.

### DESIGN SYSTEM:
- **Platform**: Web (Landing Page), Mobile-first responsive
- **Palette**: 
  - Primary: Deep Indigo (#4F46E5) - conveys wisdom and trust
  - Secondary: Warm Amber (#F59E0B) - represents inspiration and light
  - Background: Off-white (#FAFAFA) with deep navy (#0F172A) accents
  - Success/Learning: Emerald (#10B981)
- **Styles**: Softly rounded corners (rounded-2xl), whisper-soft elevations, generous whitespace

### PAGE STRUCTURE:

1. **Header**: 
   - Sticky navigation bar with glassmorphism effect
   - Logo "dotWord" on left, nav links on right (Features, About, Download)
   - Download CTA button with primary color

2. **Hero Section**:
   - High-impact headline: "Learn a New Word Every Day"
   - Subtext: "Build your vocabulary with daily words paired with inspirational quotes. Make learning a habit."
   - Two CTAs: "Download on iOS" (primary), "Learn More" (secondary/outline)
   - Feature preview: Mockup of app showing word of the day with quote

3. **Features Section**:
   - 3-column responsive card grid:
     - Daily Word notifications
     - Inspirational quotes from famous thinkers
     - Progress tracking & streaks
   - Each card with icon, title, and brief description

4. **Quote Showcase**:
   - Large, elegant typography showcasing sample quotes
   - Horizontal scroll or carousel of notable quotes
   - Subtle background pattern

5. **How It Works**:
   - 3-step visual process
   - 1. Set your daily reminder
   - 2. Receive word + quote
   - 3. Build your vocabulary

6. **App Preview Section**:
   - Side-by-side: Phone mockup showing app UI vs. stats/progress view
   - Device frame with realistic shadow

7. **Footer**:
   - GitHub link, App Store badge
   - Copyright and social links

### Visual Elements:
- Book/learning iconography
- Gradient accents (indigo to purple)
- Clean sans-serif typography (Inter or similar)
- Subtle dot grid pattern for tech credibility
- Micro-interactions on buttons (hover lift effect)
