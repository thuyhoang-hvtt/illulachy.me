# Ambient Space — Landing Page Design

## Project Context
- **Project Name**: Ambient Space
- **Type**: Web Application Landing Page
- **Tech Stack**: ReactJS
- **Purpose**: Create the perfect focus environment by mixing ambient sounds—fire crackling, forest rain, ocean waves—that boost productivity

## Enhanced Design Prompt

**Atmosphere**: Minimal, monochrome, pixel-style aesthetic. Retro computing meets digital zen—clean, focused, no distractions. Calm but intriguing, like stepping into a quiet forest at midnight.

### DESIGN SYSTEM:
- **Platform**: Web (Landing Page), Desktop-first responsive
- **Palette**: 
  - Primary: Pure Black (#000000) - stark, bold
  - Secondary: Off-White (#F5F5F5) - soft contrast
  - Accent: Dark Gray (#333333) - subtle emphasis
  - Highlight: Muted Green (#4A7C59) - nature/forest nod (sparingly)
  - Background: Near-white (#FAFAFA) or very light gray
- **Styles**: Sharp corners (no border-radius or minimal), 1px solid borders, monospace typography, pixel-perfect alignment, dithered effects where appropriate

### PAGE STRUCTURE:

1. **Header**: 
   - Fixed navigation bar with 1px bottom border
   - Logo "Ambient Space" in monospace on left, nav links on right (Sounds, About, GitHub)
   - CTA button: outlined style with border

2. **Hero Section**:
   - Headline: "Build Your Focus Soundscape"
   - Subtext: "Layer fire, rain, wind, and forest sounds to create your ideal workspace. Boost productivity with audio designed for deep work."
   - Two CTAs: "Start Mixing" (primary filled), "View on GitHub" (secondary/outline)
   - Visual: Pixel-style sound mixer preview—grid of toggle boxes with volume sliders

3. **Sound Categories Section**:
   - 4-column responsive grid:
     - **Fire** — Crackling fireplace, campfire
     - **Water** — Stream, rain, ocean waves
     - **Wind** — Gentle breeze, forest wind
     - **Nature** — Birds, forest ambience
   - Each card: minimal icon, title, one-line description

4. **Features Section**:
   - 3-column card layout:
     - **Sound Mixing** — Layer unlimited sounds to build your perfect blend
     - **Custom Presets** — Save and name your favorite combinations
     - **Focus Mode** — Minimal UI that fades when you need to concentrate
   - Cards with 1px border, no shadows

5. **Preview/Interactive Section**:
   - Simple grid of sound toggle buttons
   - Visual feedback: filled = active, outlined = inactive
   - Volume sliders (minimal, 1px lines)

6. **How It Works**:
   - 3-step vertical layout with numbers:
     - **01** — Pick your base sound
     - **02** — Layer in additional sounds
     - **03** — Enter your focus zone

7. **Footer**:
   - GitHub link
   - Minimal text copyright
   - Simple horizontal rule above

### Visual Elements:
- Monospace font (JetBrains Mono, IBM Plex Mono, or similar)
- 1px solid black borders everywhere
- Geometric pixel icons
- No rounded corners
- Grid-based layout with strict alignment
- Hover: invert colors or add 2px border
