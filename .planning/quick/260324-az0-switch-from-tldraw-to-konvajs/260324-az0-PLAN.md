<interfaces>
<!-- Current tldraw API being replaced -->
From tldraw:
- `Tldraw` component with `hideUi`, `onMount`, `shapeUtils` props
- `Editor` class with `getCamera()`, `setCamera()`, `getViewportScreenBounds()`, `createShape()`, `deleteShapes()`, `getCurrentPageShapes()`, `setCurrentTool()`, `on()`/`off()` event handlers
- `BaseBoxShapeUtil` for custom shape classes
- `HTMLContainer` for HTML content in shapes

<!-- Konva.js API to use -->
From react-konva:
- `Stage` component (container, handles pointer events)
- `Layer` component (canvas layer for shapes)
- `Group` component (grouping shapes)
- `Html` from `react-konva-utils` (HTML content in canvas)
- Stage ref for `position()`, `scale()`, `setPointersPositions()`
- `on` handlers: `onWheel`, `onDragEnd`, `onDragMove`

Camera model difference:
- tldraw: camera { x, y, z } where z=zoom, position is viewport offset
- Konva: stage { x, y, scaleX, scaleY } where position is stage offset, scale is zoom
</interfaces>

   - Implement zoom via `onWheel` handler with delta calculation:
     ```tsx
     const handleWheel = (e: KonvaEventObject<WheelEvent>) => {
       e.evt.preventDefault()
       const stage = stageRef.current
       const scaleBy = 1.1
       const oldScale = stage.scaleX()
       const pointer = stage.getPointerPosition()
       const mousePointTo = {
         x: (pointer.x - stage.x()) / oldScale,
         y: (pointer.y - stage.y()) / oldScale,
       }
       const newScale = e.evt.deltaY < 0 ? oldScale * scaleBy : oldScale / scaleBy
       const clampedScale = Math.max(ZOOM_MIN, Math.min(ZOOM_MAX, newScale))
       stage.scale({ x: clampedScale, y: clampedScale })
       stage.position({
         x: pointer.x - mousePointTo.x * clampedScale,
         y: pointer.y - mousePointTo.y * clampedScale,
       })
     }
     ```
