<objective>
Implement spaceship navigation mode where users can toggle (G key) between standard pan/zoom navigation and momentum-based spaceship flight using arrow keys.

**Purpose:** Add engaging, game-like interaction to portfolio exploration. Momentum-based physics creates satisfying "free flight" feeling with smooth camera follow. Serves as unique portfolio differentiator and demonstrates interactive development skills.

**Output:** Working game mode with:
- G key toggle (no state persistence, always starts disabled)
- Spaceship SVG cursor (48-56px, rotates to face velocity)
- Arrow key physics (acceleration → velocity → position)
- Smooth camera follow (lerp with configurable lag)
- Mauve border glow indicator when active
- Frame-rate independent physics (identical behavior 30-144 FPS)
- Standard arrow key navigation disabled during game mode
</objective>
