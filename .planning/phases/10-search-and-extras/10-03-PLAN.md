<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 2: Human verification of all Phase 10 features</name>
  <files></files>
  <action>
Present the preview server URL to the user and ask them to verify all four features:

**1. Search (SRCH-01):**
- Visit the blog index page (http://localhost:4321)
- Confirm search bar is visible below the "Writing" heading
- Type "typescript" in the search bar
- Confirm matching results appear with title and excerpt
- Clear the input — confirm results disappear

**2. Copy Button (SRCH-02):**
- Open any post with code blocks (e.g. /deep-dive-typescript or /tldraw-discovery)
- Hover over a code block — confirm a copy icon appears in the top-right corner
- Click the copy button — confirm the icon swaps to a checkmark
- Paste somewhere — confirm the code text was copied correctly
- Wait 2 seconds — confirm the icon reverts to the copy icon

**3. Table of Contents (SRCH-03):**
- On the same post page at desktop width (1280px+), confirm a "On this page" sidebar is visible to the right of the article
- Confirm it lists h2 and h3 headings (h3 entries indented under h2)
- Click a ToC entry — confirm the page scrolls to that heading
- Scroll through the post — confirm the active heading is highlighted in the ToC (mauve/interactive color)
- Resize to mobile width (below 768px) — confirm the ToC sidebar disappears completely

**4. Canvas Backlink (SRCH-04):**
- Open a post that has a matching timeline entry (check which slugs match timeline URLs)
- Confirm "See on timeline" link appears in the header metadata area (near date/reading time)
- Confirm "See on timeline" link appears in the footer (alongside "Browse all posts")
- Click the link — confirm it opens https://illulachy.me in a new tab
- Open a post that does NOT have a matching timeline entry
- Confirm the "See on timeline" link is absent from both header and footer
  </action>

  </acceptance_criteria>
