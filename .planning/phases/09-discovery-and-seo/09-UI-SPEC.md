## Spacing Scale

Declared values (from @illu/tokens 8px grid system):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`--spacing-1`) | Icon gaps, pill padding inline |
| sm | 8px (`--spacing-2`) | Inline element gaps, dot separator spacing |
| md | 16px (`--spacing-4`) | Default element spacing, card internal padding |
| lg | 24px (`--spacing-6`) | Card internal padding (`--space-card-padding`), section gaps |
| xl | 32px (`--spacing-8`) | Container horizontal padding (`--space-container-padding`) |
| 2xl | 48px (`--spacing-12`) | Between sections (`--space-breath`) |
| 3xl | 64px (`--spacing-16`) | Page-level vertical rhythm |

Exceptions:
- Touch targets: minimum 44px (`--touch-target-min`) per WCAG 2.1 ‚Äî applies to nav wordmark link and portfolio link
- Nav height: 56px tall (8px padding top + bottom, 40px content area) to maintain sticky bar proportions
- 404 page vertical centering: use `min-h-screen` with flex centering ‚Äî no spacing token applies

**Source:** `.stich/TOKENS.md` ¬ßBase Spacing

## Component Inventory

### Nav ‚Äî Sticky Glassmorphism Header (new)

- Position: `sticky top-0`, z-index `--z-sticky` (200)
- Background: `--glass-bg` (`rgba(28,28,28,0.7)`) with `backdrop-filter: blur(20px)` (`--glass-blur`)
- Border: none ‚Äî No-Line Rule applies; bottom edge defined by content separation only
- Shadow: `--glass-shadow` (`0 8px 32px rgba(0,0,0,0.2)`)
- Layout: horizontal flexbox ‚Äî `justify-between`, `items-center`
- Padding: 16px horizontal (`--spacing-4`), 12px vertical (`--spacing-3`)
- Left: wordmark "Writing" ‚Äî Noto Serif 600 20px, links to `/`, hover color `--interactive-default`, transition `--motion-hover`
- Right: "illulachy.me" ‚Äî Space Grotesk 400 16px, `--text-secondary`, links to `https://illulachy.me`, hover color `--interactive-default`, transition `--motion-hover`
- Present on: all blog pages EXCEPT 404
- **Source:** 09-CONTEXT.md D-01 through D-05

### Category Listing Page `/category/[name]` (new)

- Layout: identical to `index.astro` post list layout ‚Äî max-w prose (~65ch), mobile-first
- Heading: category name (e.g., "Engineering") ‚Äî Noto Serif 36px semibold, `--text-primary`, `--leading-tight`
- Heading margin bottom: 48px (`--spacing-12`) before post list
- Post list: reuse `<PostCard>` component exactly
- No post count in heading ‚Äî source: D-08
- **Source:** 09-CONTEXT.md D-06, D-07, D-08

### Tag Listing Page `/tag/[name]` (new)

- Identical design to category listing page
- Heading: tag name (e.g., "rust")
- Uses same template as category page
- **Source:** 09-CONTEXT.md D-07

### Category Pill (modified ‚Äî PostCard.astro)

- Change existing `<span>` to `<a href="/category/{slug}">` where slug = lowercase, spaces-to-hyphens
- Style unchanged: `bg-interactive-bg-subtle text-interactive-default rounded-full px-2 py-0.5 text-sm`
- Add hover: `hover:bg-interactive-bg-medium` transition `--motion-hover`
- **Source:** 09-CONTEXT.md D-10, code_context

### 404 Page (new)

- No navigation header
- Full-page centered layout: `min-h-screen flex flex-col items-center justify-center`
- Background: `--surface-default` (#131313)
- Heading: Noto Serif 36px semibold, `--text-primary` ‚Äî witty one-liner (see Copywriting Contract)
- Sub-line: Space Grotesk 16px regular, `--text-secondary` ‚Äî single link "Back to writing" href="/"
- Link style: `--interactive-default` color, underline on hover
- No decorative elements, no illustrations
- **Source:** 09-CONTEXT.md D-20 through D-22

## Interaction States

All interactive elements (links, nav items, category pills):

| State | Treatment |
|-------|-----------|
| Default | `--text-secondary` or `--interactive-default` per role |
| Hover | Color transitions to `--interactive-hover` (`#EAC7FF`), duration `--duration-normal` (150ms), easing `--ease-out` |
| Focus-visible | `2px solid --interactive-default`, offset 4px (already in global.css) |
| Active/pressed | `--interactive-active` (`#D197FF`) |

Post card hover: background shifts from transparent to `--surface-container-low` ‚Äî inherited from existing PostCard.astro (no change).

Nav transition on scroll: none ‚Äî glassmorphism always visible, no scroll-triggered opacity change.

## Accessibility Contract

- All links have discernible text (no icon-only links in this phase)
- Nav wordmark "Writing" ‚Äî text link, not image
- Category/tag pills that are now links must include `title` attribute with full context: `title="View all {category} posts"`
- 404 page must have `<h1>` for screen readers
- Focus-visible styles established in global.css ‚Äî no change required
- RSS link in `<head>` auto-discovered: `<link rel="alternate" type="application/rss+xml" href="/rss.xml">`

## Motion Contract

Category pill hover: background color transition, 150ms ease-out (matches existing PostCard.astro transition pattern).

No page transition animations in this phase ‚Äî Astro static navigation.
